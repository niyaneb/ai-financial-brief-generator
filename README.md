# AI Powered Financial Brief Generator

## Overview
A Python-based application that generates finanical analysis reports using AI. This project simulates how analysts create first-draft financial briefs by transforming raw company information into clear, structured insights. 
It leverages large language models (LLMs) to automate early-stage research and drafting workflows.


## Features
- Generates structured financial briefs including:
  - Summary
  - Key Insights
  - Risks
  - Opportunities
  - Final Takeaway
- Accepts raw text input (company updates, news, etc.)
- Uses AI (LLM API) for dynamic analysis
- Saves reports as `.txt` files for easy access


## How to Run
1. Open the file in Python
2. Run the program
3. Input text or article about a company
